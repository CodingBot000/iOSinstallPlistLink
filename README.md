## Tech stack & Open-source libraries
- Minimum SDK level 21
- [Kotlin] based, [Coroutines] + [Flow], [RxJava] for asynchronous.
- [Hilt] for dependency injection.
- Jetpack
  - Lifecycle 
  - ViewModel 
  - DataBinding 
  - Room Persistence 
- Architecture
  - MVVM Architecture (View - DataBinding - ViewModel - Model)
  - Repository Pattern
- [Retrofit2 & OkHttp3]
- [Glide]
